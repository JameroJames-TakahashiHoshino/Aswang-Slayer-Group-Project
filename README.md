Aswang Slayer Group Project

This project is a 2D platformer game built in Java, inspired by Filipino mythology and folklore. It features hand-drawn pixel art, animated characters, and classic platformer mechanics. This game was created as a group project, and it represents our first experience uploading a large multi-folder project to GitHub.
